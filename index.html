xmppd by ogashiwa

simple and light-weight XMPP Instant Messaging server implementation

simple and light-weight XMPP Instant Messaging server implementation

Dependencies

Python-3.2.2

Install

1. how to install # tar -xzvf xmppd-.tar.gz # cp -r ./xmppd /usr/local/ 2. start xmppd # vi /usr/local//xmppd/xmppd.conf # /usr/local//xmppd/xmppd.py \ -c /usr/local//xmppd/xmppd.conf 3. start muc # /usr/local//xmppd/muc.py \ -n -k -s ( MUST includes xmppd's name as a service domain name.) enjoy!
